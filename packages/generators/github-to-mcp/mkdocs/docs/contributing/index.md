## Quick Start

```bash
# Clone the repository
git clone https://github.com/nirholas/github-to-mcp
cd github-to-mcp

# Install dependencies
pnpm install

# Build all packages
pnpm build

# Run tests
pnpm test

# Start development server
pnpm dev
```
